# filemanager-connector-node


### Run connector

**1) Install dependencies ```npm install```**

**2) Run server ```node app.js```**



## Enviroment variables


**1) Go to the root folder of your application and create a text file called ``` .env```**

  

**2) Add the following enviromental variable:  

 - ```INITIAL_PATH``` with the desired initial directory path . Example: ``` INITIAL_PATH=/mnt/lizardfs```**
